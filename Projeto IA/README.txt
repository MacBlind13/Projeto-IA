Projeto IA

Este Trabalho tem como objetivo resolver problemas lógicos utilizando as metodologias ensinadas em sala de aula.
Para esse projeto desenvolvemos um jogo baseado no desenho animado da Disney “The owl house” e o jogo de celular Flappy Bird,
onde o propósito do jogo é que a personagem desvie dos obstáculos apresentados e consiga a maior pontuação possível utilizando 
o pygame para programção de jogos em python, e o NEAT para contrução da IA.

Referente a programação utilizada para o jogo, foi utilizado a extensão pygame, Disponível em:
Através do propt de comando de seu SO windows, ou do seu software Visual Studio Code utilize o seguinte comando: 
-pip install pygame

Referente a programação utilizada para a IA, foi utilizado a extensão NEAT, Disponível em:
Através do propt de comando de seu SO windows, ou do seu software Visual Studio Code utilize o seguinte comando: 
-pip install neat

Caso ocorra erro na hora de instalar ambas versões, tente través do propt de comando de seu SO windows,
ou do seu software Visual Studio Code utilizar o seguinte comando:
-pip install setuptools==58

Certifique-se também na na pasta diretório esteja presente o arquivo "config-feedfoward.txt", utilizado para configuração do NEAT

Nosso objetivo para o trabalho é implementar uma inteligência artificial de algoritmo genético, em específico a Neuroevolution of Augmenting Topologies (NEAT), 
para treinar automaticamente uma rede neural para jogar o jogo desenvolvido e assim, conseguir a maior quantidade de pontos possível. Esse codigo permite
também, a atribuição de novos parametros para o aprendizado da inteligência artificial, possibilitando diferentes leituras e resultados, para utilização
da ferramenta NEAT.

Através do arquivo "config-feedfoward.txt", é possível configuar as opções para utilização da inteligencia artificial da extenção NEAT.

O programa foi construido e testado utilizando a plataforma gratuita Visual Studio code para Windows, Diponível em:
-https://code.visualstudio.com/

O código fonte foi programado utilizando a extensão Python do Visual Studio Code, em sua versão 3.10.5, Disponível em:
-https://www.microsoft.com/store/productId/9PJPW5LDXLZ5
ou baixando através do próprio aplicativo Visual Studio Code.
